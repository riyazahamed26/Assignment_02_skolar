Positioning Techniques Demo

Static Positioning

This is the default positioning for HTML elements. They flow in the order they appear in the HTML document.

Relative Positioning

Allows you to position an element relative to its normal position in the document flow.

Relative Box

Absolute Positioning

Positions an element relative to its nearest positioned ancestor (or to the initial containing block if no positioned ancestor is found).

Absolute Box

Fixed Positioning

Positions an element relative to the viewport, so it stays fixed in its position even when the page is scrolled.

Fixed Box

Sticky Positioning

Acts like a relative positioned element until it reaches a certain scroll point, then it "sticks" in place.

Sticky Box
