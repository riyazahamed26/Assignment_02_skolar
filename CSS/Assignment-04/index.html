CSS Box Model

Content Box (Default)

This is a content box. The width and height include only the content, not the padding, border, or margin.

Border Box

This is a border box. The width and height include the content, padding, and border, but not the margin.

Padding

This box has padding. Padding increases the space between the content and the border.

Border

This box has a border. Border surrounds the padding and content.

Margin

This box has margin. Margin creates space outside the border, separating the box from other elements.
